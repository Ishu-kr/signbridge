Create Project

Left menu → Build → Realtime Database → Create database → Start in test mode → Enable

3

Get Config Values

⚙️ Project Settings → Your apps → click </> → Register app → Copy apiKey, databaseURL, projectId

4

Paste & Connect

Fill the 3 fields above → click Connect. Config saves in your browser — never type it again on this device!
